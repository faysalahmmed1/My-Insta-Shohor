Quetion:02 What Dose javascript Even loop do?

In JavaScript, the event loop is a fundamental mechanism that enables the asynchronous execution of code. It's an essential part of the JavaScript runtime environment, allowing the language to handle non-blocking operations efficiently. The JavaScript event loop is a fundamental concept that plays a crucial role in the asynchronous nature of JavaScript. It's responsible for handling asynchronous operations such as fetching data from a server, handling user input, and executing callbacks.


Quetion:03 Local Storage And Session Storage difference

localStorage and sessionStorage are two mechanisms provided by modern web browsers to store key-value pairs locally on the user's machine. They are both part of the Web Storage API, and they have some similarities, but there are also some key differences. (1)Scope. (2)Lifetime. (3)Storage Limit. (4)Usage. (5)Access. Special Note: sessionStorage is similar to localStorage ; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.
